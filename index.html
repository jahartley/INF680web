INF680 FT8 Home

INF680 Student Project Website

This website was created to complete the student project portion of my Fort Hays State University INF680 Network Architecture and Data Communications I course. For this project I have chosen to provide information about the FT8 digital weak signal protocol designed by Joe Taylor, K1JT, and Steve Franke, K9AN. FT8 allows digital messages to be sent and received at between amateur radio operators so they can see how far their transmissions can reach and respond to another operator thousands of miles apart.
